
# 飞书多为表格接口

## GET 列出数据表

GET /open-apis/bitable/v1/apps/{app_token}/tables

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|app_token|path|string| 是 |none|
|Authorization|header|string| 是 |none|

> 请求示例
```python
import http.client

conn = http.client.HTTPSConnection("base-api.feishu.cn")
payload = ''
headers = {
   'Authorization': 'Bearer pt-eHTfXM5N-Z_DP5H4UdggH4sjyPpfe-2cYHO0kmiYAQAAHABAsAKAC47s_BN9',
   'User-Agent': 'Apifox/1.0.0 (https://apifox.com)'
}
conn.request("GET", "/open-apis/bitable/v1/apps/KnOFbTu1harM6wsNbWdcro2Rnoc/tables", payload, headers)
res = conn.getresponse()
data = res.read()
print(data.decode("utf-8"))
```

|名称|位置|类型|必选|说明|

|app_token|path|string| 是 |none|
